# Bootstrap Badge Component Activity

This repository demonstrates how to use the Bootstrap Badge component.

## What are Badges?

Badges are small labels used to show notification counts or categories.

Example uses:
	•	unread messages
	•	notifications
	•	assignment counts
	•	category tags
	•	user status indicators

Bootstrap Classes Used

.container → centers the page content and adds responsive spacing

.badge → creates a badge label

.badge-primary → applies the primary color to a badge

.badge-success → applies a green success color to a badge

.badge-warning → applies a yellow/orange warning color to a badge

.badge-danger → applies a red danger/error color to a badge
