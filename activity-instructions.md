# Bootstrap Badges Activity

## Scenario
You are building a school dashboard website for students.  
The dashboard contains sections like Messages, Assignments, and Notifications.

To help students quickly see updates, you must use Bootstrap badges to show the number of new items.

## Task

Work in pairs and create a webpage that includes:

• A button for Messages  
• A button for Assignments  
• A button for Notifications  

Next, add Bootstrap badges beside each button to show the number of new items.

Example:

Messages → 3 new messages  
Assignments → 1 new assignment  
Notifications → 5 notifications

## Goal
Use the Bootstrap badge component to display notification counts.
